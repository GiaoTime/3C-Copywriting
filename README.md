# 3C数码详情页文案生成器 📝

[![GitHub](https://img.shields.io/badge/GitHub-3C--Copywriting-blue?style=flat&logo=github)](https://github.com/GiaoTime/3C-Copywriting)
[![Skill Type](https://img.shields.io/badge/Type-AI%20Agent%20Skill-green?style=flat)](.)
[![Category](https://img.shields.io/badge/Category-3C%20E--commerce-orange?style=flat)](.)

> **专注3C数码品类的爆款详情页文案生成器**  
> 通过结构化信息采集 + 竞品对比分析，输出苹果/乔布斯风格的科技风长文案

---

## 🎯 核心功能

| 功能模块 | 说明 |
|---------|------|
| **信息采集** | 20项结构化问卷，覆盖产品基础、技术配置、差异化卖点 |
| **竞品分析** | 自动检索京东/天猫/拼多多/抖音商城竞品数据 |
| **文案生成** | 苹果风格长文案，含配图建议、数据高亮、购买理由 |
| **多平台适配** | 支持电商详情页、社媒种草、直播脚本等多场景 |

### 工作流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  用户触发   │ → │ 信息采集问卷 │ → │ 信息确认检查点 │ → │ 竞品补充检索 │
└─────────────┘    └─────────────┘    └─────────────┘    └──────┬──────┘
                                                                  │
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│ 输出Markdown │ ← │ 生成苹果风格文案 │ ← │ 文案预览确认检查点│ ← │ 竞品对比分析 │
└─────────────┘    └─────────────┘    └─────────────┘    └──────┬──────┘
                                                                  │
                                            ┌─────────────────────┘
                                            │
                                    ┌─────────────┐
                                    │ 竞品选择确认检查点│
                                    └─────────────┘
```

### 检查点设计

为防止自主失控，确保用户全程参与关键决策：

1. **信息确认检查点** - 确保产品信息准确无误
2. **竞品选择确认检查点** - 确保竞品对比合理有效  
3. **文案预览确认检查点** - 确保文案方向符合用户预期

### 信息采集问卷（20题）

<details>
<summary>点击展开完整问卷</summary>

#### 基础信息
1. 产品名称型号？（如：iPhone 16 Pro）
2. 产品类别？（手机/平板/笔记本/耳机/智能手表/相机/其他）
3. 品牌定位？（高端旗舰/性价比/年轻化/商务）
4. 核心售价？有无促销价？
5. 上市状态？（已上市/预售/即将发布）

#### 技术配置
6. 核心处理器/芯片型号？
7. 内存/存储配置版本？
8. 屏幕参数（尺寸/分辨率/刷新率/材质）？
9. 电池容量/充电功率/续航时间？
10. 机身材质/工艺/重量厚度？
11. 影像系统配置？（如适用）
12. 有无独家技术/专利/行业首创？

#### 差异化卖点
13. 相比上代最大3个升级点？
14. 相比竞品最突出的差异化？
15. 有无行业认证/设计奖项/权威评测？

#### 竞品信息
16. 主要竞品品牌和型号？（2-3个）
17. 竞品价格区间？

#### 目标人群
18. 核心用户画像（年龄/职业/使用场景）？
19. 用户痛点？
20. 购买动机？

</details>

## ✨ 文案风格特色

### 🍎 苹果风格要点

| 原则 | 示例 |
|-----|------|
| **极简短句** | 3-8字主标题，一句话定义产品 |
| **动词优先** | 强动词开头，直击用户利益 |
| **数据说话** | "26小时视频播放" 替代 "超长续航" |
| **对比思维** | "像素密度提升25%" 替代 "屏幕清晰" |
| **技术诗意** | "A17 Pro。游戏主机级性能，随手可得。" |

### 🏷️ 高亮标注

- 🔥 **行业顶尖/首创/突破性数据**
- ✓ **明显优于竞品的数据**

### 📸 配图建议

每个卖点后附带具体配图方向：
- 产品特写
- 场景图
- 对比图
- 爆炸图
- 动效示意

## 📦 输出示例

生成的文案包含以下模块：

```markdown
# 【产品名】详情页文案

## 主标题
一句话定义产品（3-8字，极简有记忆点）

## 副标题  
一句话价值主张（15-25字）

---

## 卖点1：[3-5字主标题]
**副标题**：一句话解释

**数据支撑**：
- 参数1：[数值] 🔥
- 参数2：[数值]
- 对比：比竞品X强[XX]%

**配图建议**：[具体方向]

---

## 配置总览表
| 配置项 | 参数 | 行业水平 |
|-------|------|---------|
| 处理器 | XXX | 🔥 旗舰级 |

## 竞品对比
（表格+优势总结）

## 目标人群
（画像+痛点+解决方案）

## 购买理由
3句话总结
```
