# 苹果风格文案指南

## 核心原则

### 1. 极简主义
- **短句为王**：每句话不超过15字
- **段落精简**：每段不超过3行
- **去掉冗余**：删除所有"的"、"了"、"和"等连接词

### 2. 动词优先
- **强动词开头**：用动词直接表达价值
- **主动语态**：避免被动语态
- **动作导向**：描述用户能做什么

### 3. 数据说话
- **具体数字**：不用模糊形容词
- **对比数据**：显示提升百分比
- **真实数据**：基于实际测试

### 4. 对比思维
- **竞品对比**：不说"很好"，说"比X好多少"
- **代际对比**：显示升级幅度
- **行业对比**：在行业中的位置

### 5. 技术诗意
- **参数翻译**：把技术参数翻译成用户价值
- **场景连接**：技术参数在具体场景中的作用
- **情感共鸣**：技术带来的情感体验

## 文案结构模板

### 主标题（3-8字）
- 一句话定义产品
- 极简有记忆点
- 避免形容词

**示例**：
- 重新定义，旗舰。
- 影像。无界。
- 大，有所为。

### 副标题（15-25字）
- 一句话价值主张
- 包含核心卖点
- 引导继续阅读

**示例**：
- A18 Pro芯片，5倍光学变焦，钛金属机身。iPhone的又一次飞跃。
- 徕卡Summilux光学系统，遇见双向卫星通信。6499元起。
- 13.2英寸OLED平板，星闪技术重新定义创作边界。

### 卖点模块结构
```
## 卖点标题：[3-5字]
**副标题**：一句话解释

**数据支撑**：
- 参数1：[数值] 🔥
- 参数2：[数值]
- 对比：比竞品X强[XX]%

**配图建议**：[具体方向]
```

## 数据标注规范

### 🔥 绝对优势标注
使用场景：
1. 行业首创技术
2. 突破性数据
3. 独家功能
4. 权威认证第一

**示例**：
- 连接延迟：0ms 🔥
- 屏幕亮度：3000尼特 🔥
- 抗摔性能：提升10倍 🔥

### ✓ 相对优势标注
使用场景：
1. 明显优于竞品
2. 代际显著提升
3. 价格优势明显
4. 配置领先

**示例**：
- 内存：12-16GB ✓
- 重量：580g ✓
- 价格：比竞品低30% ✓

## 配图建议规范

### 配图类型
1. **产品特写**：突出设计细节
2. **场景图**：展示使用场景
3. **对比图**：直观显示优势
4. **爆炸图**：展示内部结构
5. **动效示意**：展示交互效果

### 配图方向具体化
- ❌ "产品图片"
- ✅ "45度角产品展示，突出钛金属边框光泽"
- ✅ "户外夜景拍摄对比，显示低光表现优势"
- ✅ "与竞品厚度对比侧视图"

## 常见错误修正

### 模糊表述修正
- ❌ "超长续航" → ✅ "26小时视频播放"
- ❌ "屏幕清晰" → ✅ "像素密度提升25%"
- ❌ "拍照效果好" → ✅ "夜景进光量提升2倍"

### 技术参数翻译
- ❌ "A17 Pro芯片，6核GPU" → ✅ "A17 Pro。游戏主机级性能，随手可得。"
- ❌ "1英寸大底传感器" → ✅ "1英寸。专业相机级感光元件，手机摄影新标准。"
- ❌ "120Hz刷新率" → ✅ "120Hz。每一次滑动，都如丝般顺滑。"

### 价值主张表达
- ❌ "我们的产品很好" → ✅ "选择X，就是选择Y的未来标杆。"
- ❌ "功能强大" → ✅ "在X生态中，每一次交互都自然流畅。"
- ❌ "性价比高" → ✅ "Y元起，享受比Z元旗舰更快的充电、更坚固的防护。"

## 文案检查清单

### 结构检查
- [ ] 主标题3-8字
- [ ] 副标题15-25字
- [ ] 卖点数量3-6个
- [ ] 每个卖点有数据支撑
- [ ] 每个卖点有配图建议

### 风格检查
- [ ] 无模糊形容词
- [ ] 动词优先开头
- [ ] 具体数字代替模糊词
- [ ] 有竞品对比数据
- [ ] 技术参数翻译成用户价值

### 标注检查
- [ ] 绝对优势用🔥标注
- [ ] 相对优势用✓标注
- [ ] 标注位置正确
- [ ] 标注理由充分
